# My Awesome React Native Expo App

Esta es una aplicación móvil desarrollada con React Native y Expo que utiliza la cámara del dispositivo para tomar fotos y muestra las imágenes guardadas en la galería.

## Instalación

1. Clona este repositorio en tu máquina local.

```bash
git clone https://github.com/Anderson1703/tecnical-test-edynamicslog.git
```

2. Ve al directorio del proyecto.

```bash
cd tecnical-test-edynamicslog
```

3. Instala las dependencias utilizando npm o yarn.

```bash
npm install
```
o
```bash
yarn install
```

4. Asegúrate de tener Expo CLI instalado globalmente. Si no lo tienes, puedes instalarlo con el siguiente comando:

```bash
npm install -g expo-cli
```

## Ejecución

1. Inicia la aplicación con Expo.

```bash
expo start
```

2. Escanea el código QR generado con la aplicación Expo Go en tu dispositivo móvil.

3. ¡Disfruta de la aplicación!

## Funcionalidades

### Pantalla de Cámara

- La pantalla de cámara te permite tomar fotos.
- Puedes cambiar entre la cámara frontal y trasera.
- Controla el flash de la cámara.
- Activa el modo HDR para tomar fotos de alta calidad.

### Pantalla de Imágenes

- La pantalla de imágenes muestra las fotos guardadas en la galería del dispositivo.
- Puedes hacer clic en una imagen para verla en tamaño completo.

## Dependencias Principales

- `react-native-vision-camera`: Utilizado para acceder a las funcionalidades de la cámara del dispositivo.
- `@react-native-community/hooks`: Utilizado para acceder al estado de la aplicación.
- `@react-native-camera-roll/camera-roll`: Utilizado para acceder a la galería de imágenes del dispositivo.
- `react-native-modal`: Utilizado para mostrar las imágenes en un modal.
